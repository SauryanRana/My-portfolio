AI Projects

I am passionate about artificial intelligence and the endless possibilities it presents. I have used Python to build a variety of AI projects. Hope you enjoy exploring it and learning more about my journey in the world of AI.

Face Detection

In this project, I utilised the OpenCV library and Haar cascades to create a face detection system. Haar cascades are machine learning classifiers that have been trained to recognize specific features in images, such as faces. By using pre-trained Haar cascades, I was able to enhance the performance and accuracy of my system. This project has numerous potential applications, including security systems, social media, and human-computer interaction.

Binary and Multiple Image Classifier using Kaggle

In this project, I used Kaggle to train and test a binary image classifier to distinguish between images of cats and dogs. I obtained a dataset of cat and dog images from Kaggle and used a variety of techniques, such as data preprocessing and feature engineering, to improve the performance of the classifier. The classifier was able to achieve high accuracy in identifying and correctly classifying images of cats and dogs. This project has potential applications in the fields of pet identification and animal classification.

Object Recognition

In this project, I collected and labeled a dataset of images containing the objects I wanted to recognize, and then trained a cascade classifier using this dataset. Once the classifier was trained, I was able to use it to detect objects in an image or video, and display the results by highlighting the objects or displaying the object labels. Object recognition can be used in a wide range of applications, including video surveillance, image search, and autonomous vehicles.
